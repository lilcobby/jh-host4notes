## Description

The application can be used to generate notes based on user input, save those notes, and a user can select from previous notes or delete previous notes.

## Installation

Install node.js. navigate to my github profile, linked below, and clone the jh-host4notes repo to your machine. npm install, i express, i uniqid. i path.

## Usage

To test the application, navigate to the server.js file in your terminal and start the server with npm run start.

now navigate to http://localhost:3001 and you will be directed to the home page. http://....3001/notes will navigate to the notes page. now enter some notes and save them or delete them as needed.

or navigate to https://jh-host4notes-0cce13beaae8.herokuapp.com/ to test the application on heroku.
